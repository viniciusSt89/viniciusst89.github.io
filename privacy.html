Política de Privacidade - SmartVendasPRO

Desenvolvedor: VS Apps Studio

E-mail de Suporte: suporte.vsapps@gmail.com

Última atualização: 2025

1. Informações Coletadas

O SmartVendasPRO coleta apenas informações necessárias para o funcionamento do aplicativo. Dados financeiros e informações inseridas pelo usuário ficam armazenados localmente no dispositivo.

1.1. Informações fornecidas voluntariamente

Nomes, valores, vendas, compras, parcelas, juros e foto de perfil.

1.2. Login opcional

Se o usuário optar por login via Google/Apple, coletamos: ID, nome e e-mail para identificar conta PRO e usar backup na nuvem.

1.3. Dados automáticos de terceiros

Serviços como Firebase, Google Drive API e AdMob podem coletar informações técnicas, tais como IP, modelo do dispositivo, dados de uso e identificadores para anúncios.

2. Permissões

O app utiliza permissões como Armazenamento, Galeria e Internet para exportar arquivos, selecionar fotos e realizar backups.

3. Uso das Informações

Utilizamos os dados para funcionamento do app, geração de relatórios, backups e verificação de assinatura PRO.

4. Armazenamento

Os dados estão armazenados localmente no dispositivo e, opcionalmente, no Google Drive do usuário. Nada é enviado aos servidores do desenvolvedor.

5. Compartilhamento

Não compartilhamos dados pessoais com terceiros, exceto serviços essenciais como Google, Firebase e AdMob.

6. Direitos do Usuário (LGPD/GDPR)

O usuário pode acessar, editar ou excluir seus dados a qualquer momento através do próprio app.
